Apliction - Создание / Изменение нескольких заявок
POST запрос на создание нескольких заявок. Для этого вам нужны как минимум api ключ, клиент и сами заявки.
https://app.relog.kz/api/v2/applications
Пример запроса.
curl -X POST \
https://app.relog.kz/api/v2/applications \
-H 'content-type: application/json' \
-d '{
"api_key": "xxxxxxxxxxxxxx",
"options": {
"multi": true
},
"data": [{
"client": {
"phone": "7702xxxxxxx",
"allPhones": ["7702xxxxxx1","7702xxxxxx2"],
"name": "clientName",
"email": "email@example.com",
"description": "Some description"
},
"application": {
"externalId": 1234,
"additionalDetails": "some details",
"details": "other details",
"price": 0,
"planDeliveryPeriod": {
"startDate": 1492590418399,
"endDate": 1492590418399
}
]
}]
}'
